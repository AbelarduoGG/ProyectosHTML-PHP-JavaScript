Sublime Surge frente a objetos de grandes dimensiones, desmesurados, desmadrados, ilimitados, peligrosos y hasta potencialmente destructivos; y ante creaciones trágicas, conmovedoras y estremecedoras. Muy liviano, fácil de instalar y una versión portable. Es totalmente configurable, pero si queremos una configuración avanzada es un poco difícil de hacer para usuarios novatos. Sublime Documentación
Brackets Es un editor de código que soporta distintos lenguajes, cuenta con temas para poner de color la letra según la estructura del código. Son muy económicos respecto al resto y ofrecen una gran durabilidad, funcionalidad y capacidad de adhesión. Estéticamente son los menos agraciados. Brackets Documentación
Atom Git y GitHub integrados. Autocompletado. Atajos de teclado Atom te ayuda a escribir el código de forma rápida y flexible a través de su inteligente modo de autocompletado Debido a su falta de potencia, vinculada de forma directa con la búsqueda del ahorro de energía y de la reducción de los costes, los procesadores Intel Atom no son capaces de mover los juegos más avanzados. Atom Documentación
Notepad ++ Liviano. Autocompletado. Permite localizar con rapidez. Nos permite localizar y editar con rapidez la línea aludida. · No sirve para hacer grandes existos de textos. Notepad ++ Documentación
Visual Studio Code Es gratuito y de código abierto. Es altamente personalizable. Tiene soporte nativo para gran variedad de lenguajes, entre ellos podemos destacar los principales del desarrollo Web: HTML, CSS, y JavaScript. Puede ser lento en proyectos grandes. Requiere conocimientos técnicos para personalizar. Problemas de estabilidad. Visual Studio Code Documentación
Netbeans Proporciona una base modular y extensible para el desarrollo de aplicaciones. Ayuda en el desarrollo de cualquier tipo de software y en sus distintas fases. Poca disponibilidad de plugins. Netbeans Documentación
Komodo Es un excelente editor de código similar a Notepad ++- Herramienta eficiente con las típicas funciones, una interfaz cómoda y un conjunto compacto de funciones. Este editor gratuito es útil para desarrollar código sencillo pero las funciones que ofrece generalmente son limitadas. Komodo Documentación



Servidores web
Servidor Características Ventajas Desventajas Sitio web oficial Documentación en línea
Apache Es un servidor web de de código abierto, multiplataforma y gratuito. De código abierto y gratuito, con una gran comunidad de usuarios. Presenta problemas de estabilidad por encima de 10, 000 conexiones. Apache Documentación
IIS Es modular para permitir una gestión ordenada y la posbilidad de agregar funciones adicionales. Facilidad de instalación, configuración y uso. Este servidor no es multiplataforma, solo funciona bajo Windows. IIS Documentación
Nginx Servidor de archivos estáticos, índices y autoindexado. Se trata de un software multiplataforma, por lo tanto podremos instalarlo en la mayoría de nuestros servidores. No es tan compatible con aplicaciones web que usen contenido dinámico. Nginx Documentación
LiteSpeed Es uno de los servidores que ofrece la posbilidad de alojar en él sitios y páginas web con todo su contenido. Hasta 9 veces más rápido que Apache. Aumenta la escalabilidad. Se trata de un software propietario y hay que pagar una licencia para poder usarlo. LiteSpeed Documentación
OpenResty Es un servidor web, con una arquitectura basada en eventos basado en Nginx y LuaJIT. Cualquier persona lo puede utilizar. Transparencia. Licencia aplicada. Si estás decidido a implementar un open source, te aconsejamos que integres a un especialista de TI a tu equipo. La apertura del código y la ausencia de un sistema de protección puede perjudicar la seguridad de tus datos. OpenResty Documentación
